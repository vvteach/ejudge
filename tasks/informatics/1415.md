# Задача №1415. Шифр Юлия

Юлий Цезарь использовал свой способ шифрования текста. Каждая буква заменялась на следующую
по алфавиту через *K* позиций по кругу. Необходимо по заданной шифровке определить исходный текст.
### Входные данные
В первой строке дана шифровка, состоящая из заглавных латинских букв. Во второй строке
число *K* (1 ≤ *K* ≤ 10).
### Выходные данные
Требуется вывести результат расшифровки.
### Примеры
**Входные данные:**
```
XPSE
1
```
**Выходные данные:**
```
WORD
```
**Входные данные:**
```
ZABC
3
```
**Выходные данные:**
```
WXYZ
```
