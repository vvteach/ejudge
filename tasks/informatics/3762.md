# Задача №3762. Самое частое слово

Дан текст (строк может быть много). Выведите слово, которое в этом тексте встречается чаще всего.
Если таких слов несколько, выведите то, которое меньше в лексикографическом
порядке.
### Входные данные
Вводится текст.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
apple orange banana banana orange
```
**Выходные данные:**
```
banana
```
