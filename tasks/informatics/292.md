# Задача №292. Максимум из двух чисел

### Входные данные
Даны два целых числа, каждое число записано в отдельной строке.
### Выходные данные
Выведите наибольшее из данных чисел.
### Примеры
**Входные данные:**
```
1
2
```
**Выходные данные:**
```
2
```
