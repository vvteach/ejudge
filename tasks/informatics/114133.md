# Задача №114133. Дробное остовное дерево

Вам дан связный граф с $n$ вершинами пронумерованными от $0$ до $n - 1$ и $m$ ребрами. На каждом ребре записана пара чисел $a\_{i}$, $b\_{i}$.
Вам нужно найти такое остовное дерево этого графа, что значение выражения $\frac{\sum a\_{i}}{\sum b\_{i}}$ при суммировании по ребрам остовного дерева принимает максимальное значение.
В качестве ответа выведите значение выражения в виде несократимой рациональной дроби $\frac{p}{q}$, если эта величина целая, то $q$ должно быть равно $1$.
### Входные данные
Первая строка содержит два числа $n$ и $m$ ($2 \leq n \leq 300\,000$, $n - 1 \leq m \leq 300\,000$).
На следующих $m$ строках записано по четыре числа $u\_{i}, v\_{i}, a\_{i}, b\_{i}$ ($0 \leq u\_i, v\_i \leq 300\,000, 1 \leq a, b \leq 100$), задающие ребро между вершинами $u\_{i}$ и $v\_{i}$, с записанными на нем числами $a\_{i}, b\_{i}$.
### Выходные данные
Выведите ответ в формате $\frac{p}{q}$, где $p$ и $q$ взаимно простые числа. Если $\frac{p}{q}$ — целое, $q$ должно быть равно $1$.

**Выходные данные:**
```
2/3
```
