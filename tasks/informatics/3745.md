# Задача №3745. Удаление символа

Дана строка. Удалите из этой строки все символы `@`.
### Входные данные
Вводится строка.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
Bilbo.Baggins@bagend.hobbiton.shire.me
```
**Выходные данные:**
```
Bilbo.Bagginsbagend.hobbiton.shire.me
```
