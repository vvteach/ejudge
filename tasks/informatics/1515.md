# Задача №1515.

Найдите сумму цифр введенного целого числа. (На каждом шаге выделяется последняя цифра числа, а затем число делится на 10. Процесс повторяется, пока число не станет равным 0).
### Входные данные
На вход программе подается целое неотрицательное число n ≤ $10^9$.
### Выходные данные
Выведите одно целое число, являющееся ответом на задачу.
### Примеры
**Входные данные:**
```
1234
```
**Выходные данные:**
```
10
```
