# Задача №436. Принадлежность точки отрезку

### Входные данные
Вводятся шесть чисел - координаты точки и координаты концов отрезка.
### Выходные данные
Выведите одну строку "YES", если точка принадлежит отрезку, и "NO" в противном случае.
### Примеры
**Входные данные:**
```
4 0 4 2 4 5
```
**Выходные данные:**
```
NO
```
