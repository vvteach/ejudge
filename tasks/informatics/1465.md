# Задача №1465. Разрезание на квадраты

Полоска бумаги имеет размеры *A* × *B*. Каждый раз от нее отрезается квадрат
максимального размера до тех пор, пока не получится квадрат. Сколько квадратов получится?
### Входные данные
Программе даны числа *A* и *B* (1 ≤ *A*, *B* ≤ 109).
### Выходные данные
Требуется вывести количество квадратов.
### Примеры
**Входные данные:**
```
15 3
```
**Выходные данные:**
```
5
```
**Входные данные:**
```
12 8
```
**Выходные данные:**
```
3
```
**Входные данные:**
```
5 5
```
**Выходные данные:**
```
1
```
