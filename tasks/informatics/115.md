# Задача №115. Количество нулей

Дано натуральное число N. Найдите количество нулей среди всех цифр числа N.
### Входные данные
Задано единственное число N
### Выходные данные
Необходимо вывести количество нулей среди всех цифр числа N.
### Примеры
**Входные данные:**
```
100
```
**Выходные данные:**
```
2
```
