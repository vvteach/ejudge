# Задача №112179. Римская система

Напишите программу, которая выводит запись переданного ей числа в римской системе счисления. Используйте процедуру.
### Входные данные
Входная строка содержит натуральное число
*N*
(
1 ≤
*N*
≤ 3999
).
### Выходные данные
Программа должна вывести запись числа
*N*
в римской системе счисления.
### Примеры
**Входные данные:**
```
1234
```
**Выходные данные:**
```
MCCXXXIV
```
