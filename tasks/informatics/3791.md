# Задача №3791. Длина отрезка

Даны четыре действительных числа: $x\_1$, $y\_1$, $x\_2$, $y\_2$. Напишите функцию
`distance(x1, y1, x2, y2)`, вычисляющую расстояние
между точкой $(x\_1,y\_1)$ и $(x\_2,y\_2)$. Считайте четыре действительных числа
и выведите результат работы этой функции.
### Входные данные
Вводятся четыре действительных числа.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
0
0
1
1
```
**Выходные данные:**
```
1.41421
```
