# Задача №1120. 0-1 рюкзак: точный вес

Дано *N* золотых слитков массой *m*1, …, *mN*. Ими наполняют рюкзак, который выдерживает вес не более *M*. Можно ли набрать вес в точности *M*?
### Входные данные
В первой строке вводится натуральное число *N*, не превышающее 100 и натуральное число *M*, не превышающее 10000.
Во второй строке вводятся *N* натуральных чисел *mi*, не превышающих 100.
### Выходные данные
Выведите YES или NO.
### Примеры
**Входные данные:**
```
1 5968
18
```
**Выходные данные:**
```
NO
```
**Входные данные:**
```
4 102
50 52 54 2
```
**Выходные данные:**
```
YES
```
