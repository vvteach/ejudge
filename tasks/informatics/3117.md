# Задача №3117. Количество слов

Дана строка, состоящая из слов, разделенных пробелами. Определите, сколько в ней слов. Используйте для решения задачи метод count.
### Входные данные
Вводится строка.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
Hello world
```
**Выходные данные:**
```
2
```
