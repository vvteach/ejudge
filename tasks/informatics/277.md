# Задача №277. Расстояние от точки до прямой

### Входные данные
Пять чисел – координаты точки и коэффициенты A, B и C уравнения прямой.
### Выходные данные
Одно число – расстояние от точки до прямой.
### Примеры
**Входные данные:**
```
1 5 0 -4 8
```
**Выходные данные:**
```
3.00000
```
