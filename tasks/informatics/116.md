# Задача №116. Минимальная и максимальная цифры

Дано натуральное число N. Необходимо вывести наименьшую и наибольшую цифры данного числа через пробел.
### Входные данные
Задано единственное число N
### Выходные данные
Необходимо вывести наименьшую и наибольшую цифры данного числа через пробел.
### Примеры
**Входные данные:**
```
37
```
**Выходные данные:**
```
3 7
```
