# Задача №132. Сложение

Напишите программу, которая вычисляет сумму двух целых неотрицательных чисел (длина каждого из чисел не превосходит 250)
### Входные данные
Даны два числа, каждое записано на отдельной строке.
### Выходные данные
Выведите сумму данных чисел.
### Примеры
**Входные данные:**
```
3
5
```
**Выходные данные:**
```
8
```
