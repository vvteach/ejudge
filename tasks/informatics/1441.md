# Задача №1441. МегаНОД

Дано N чисел. Найти самое большое число, на которое делятся все N чисел.
### Входные данные
В первой строке дано число N. Во второй строке даны через пробел N чисел (1 <= N <= 1000).
### Выходные данные
Выведите искомое число
### Примеры
**Входные данные:**
```
1
3
```
**Выходные данные:**
```
3
```
**Входные данные:**
```
3
9 15 22
```
**Выходные данные:**
```
1
```
