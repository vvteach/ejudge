# Задача №3542. Диофантово уравнение - 1

Даны числа a, b, c, d. Выведите в порядке возрастания все целые числа
от 0 до 1000, которые являются корнями уравнения
ax3+bx2+cx+d=0.
### Входные данные
Вводятся целые числа a, b, c и d.
### Выходные данные
Выведите ответ на задачу. Если в указанном промежутке нет корней уравнения, то ничего выводить не нужно.
### Примеры
**Входные данные:**
```
-1
1
-1
1
```
**Выходные данные:**
```
1
```
**Входные данные:**
```
1
1
1
1
```
