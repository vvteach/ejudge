# Задача №278. Расстояние от точки до луча

### Входные данные
Шесть чисел – координаты точки и координаты начала и конца вектора.
### Выходные данные
Одно число – расстояние от точки до луча, определяемого вектором.
### Примеры
**Входные данные:**
```
2 3 0 0 4 0
```
**Выходные данные:**
```
3.00000
```
**Входные данные:**
```
-1 1 0 0 4 0
```
**Выходные данные:**
```
1.41421
```
