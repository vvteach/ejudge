# Задача №114333. Время - деньги!

Компания NetLine хочет предложить широкополосный интернет в
$N$ городах. Для этого достаточно построить сеть из $N-1$
широкополосных линий связи между городами с тем свойством, чтобы
сообщение могло дойти из любого города в любой другой город в этой
сети. NetLine уже определила все пары городов, между которыми
можно построить линию. Для каждой такой линии известны стоимость и
время, которое потребуется для создания линии.
Компания заинтересована в минимизации как общего количества
времени (линии создаются одна за другой), так и общей суммы денег,
затраченных на создание всей сети. Компания решила использовать
следующую формулу для оценки стоимости сети:
$SumTime$ = суммарное время, потраченное на создание
выбранных линий
$SumMoney$ = сумма денег, потраченных на построение выбранных линий
$V = SumTime \* SumMoney$
Предъявите список из $N-1$ линий, который минимизирует
значение $V$.
### Входные данные
Первая строка ввода содержит целые числа $N$ и $M$ -
количество городов и количество пар городов, которые можно
соединить. Города нумеруются с 0.
Каждая из следующих $M$ строк содержит четыре целых числа
$x$, $y$, $t$ и $c$ – означающие, что город
$x$ может быть соединён с городом $y$ за время $t$ со
стоимостью $c$.
$1 \le N \le 200$
$1 \le M \le 10 000$
$0 \le x,y \le N-1$
$1 \le t,c \le 255$
### Выходные данные
В первой строке выведите два числа: общее время $(SumTime)$ и
общее количество денег $(SumMoney)$, использованные в
оптимальном решении (с минимальным значением $V$).
Далее в $N-1$ строках выведите линии, которые будут
построены. Каждая строка должна содержать пару чисел $(x, y)$,
описывающих линию для построения (она должна быть среди линий,
описанных во входном файле). Пары могут быть распечатаны в любом
порядке. Если существует несколько решений, можно вывести любое из них.
### Примеры
**Входные данные:**
```
5 7
0 1 161 79
0 2 161 15
0 3 13 153
1 4 142 183
2 4 236 80
3 4 40 241
2 1 65 92
```
**Выходные данные:**
```
279 501
2 1
0 3
0 2
3 4
```
