# Задача №112213. Сумма цифр

Напишите программу, которая считает сумму цифр введённого числа.
### Входные данные
Входная строка содержит одно натуральное число.
### Выходные данные
Программа должна вывести сумму цифр введённого числа.
### Примеры
**Входные данные:**
```
12345
```
**Выходные данные:**
```
15
```
