# Задача №3155. Больше предыдущего

Дан список чисел. Выведите все элементы списка, которые больше предыдущего элемента.
### Входные данные
Вводится список чисел. Все числа списка находятся на одной строке.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1 5 2 4 3
```
**Выходные данные:**
```
5 4
```
