# Задача №1446. Результаты олимпиады

Во время проведения олимпиады каждый из участников получил свой идентификационный номер – натуральное
число. Необходимо отсортировать список участников олимпиады по количеству набранных ими баллов.
### Входные данные
На первой строке дано число *N* (1 ≤ *N* ≤ 1000) – количество участников.
На каждой следующей строке даны идентификационный номер и набранное число баллов
соответствующего участника. Все числа во входном файле не превышают 105.
### Выходные данные
В выходной файл выведите исходный список в порядке убывания баллов. Если у некоторых участников
одинаковые баллы, то их между собой нужно упорядочить в порядке возрастания идентификационного
номера.
### Примеры
**Входные данные:**
```
3
101 80
305 90
200 14
```
**Выходные данные:**
```
305 90
101 80
200 14
```
**Входные данные:**
```
3
20 80
30 90
25 90
```
**Выходные данные:**
```
25 90
30 90
20 80
```
