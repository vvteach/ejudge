# Задача №271. Уравнение прямой II

### Входные данные
Четыре числа – координаты точки на прямой и координаты вектора нормали к этой прямой.
### Выходные данные
Три числа – коэффициенты A, B и C уравнения этой прямой.
### Примеры
**Входные данные:**
```
0 0 1 1
```
**Выходные данные:**
```
1 1 0
```
