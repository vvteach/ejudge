# Задача №3642. Список квадратов

По данному целому числу N распечатайте все квадраты натуральных чисел,
не превосходящие N, в порядке возрастания.
### Входные данные
Вводится натуральное число.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
50
```
**Выходные данные:**
```
1 4 9 16 25 36 49
```
