# Задача №1406. Анаграммы

Слово называется анаграммой другого слова, если оно может быть получено перестановкой его символов.
### Входные данные
Даны два слова на отдельных строках. Слова состоят из строчных латинских букв и цифр.
Длины слов не превышают 255.
### Выходные данные
Требуется вывести "YES"  – если введенные слова являются анаграммами друг друга,
"NO"  – если нет.
**Примечание**
Сложность работы программы должна быть *O(n).***Использование встроенной сортировки(sort, sorted), алгоритмов сортировки пузырёк/quick sort/merge sort и других запрещено!**
### Примеры
**Входные данные:**
```
sharm
marsh
```
**Выходные данные:**
```
YES
```
**Входные данные:**
```
ananas
nnaass
```
**Выходные данные:**
```
NO
```
