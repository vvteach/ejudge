# Задача №317. Число сочетаний

По данным натуральным *n* и *k* вычислите значение $C\_n^k = \frac{n!}{k!(n-k)!}$ (число сочетаний из *n* элементов по *k*).
### Входные данные
Вводятся 2 числа - *n* и *k* ($n, k \leq 10$).
### Выходные данные
Необходимо вывести  значение $C\_n^k$.
### Примеры
**Входные данные:**
```
2
1
```
**Выходные данные:**
```
2
```
