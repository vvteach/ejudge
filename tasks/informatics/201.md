# Задача №201. Числа Фибоначи

F1 = *F2* = 1, Fn = Fn-1 + Fn-2, при n > 2
### Входные данные
В единственной строке входных данных записано натуральное число *n* (1*≤n*≤45).
### Выходные данные
Вывести одно число Fn
### Примеры
**Входные данные:**
```
2
```
**Выходные данные:**
```
1
```
**Входные данные:**
```
5
```
**Выходные данные:**
```
5
```
