# Задача №1448. Апельсины бочками

Бизнесмен Василий после прочтения известной книги решил открыть новый бизнес – отгружать апельсины бочками.
Партнерам важно знать сколько именно бочек апельсинов отгружается каждый день.
Мобильный телефон Василия поддерживает только транслит, поэтому он передает сообщения вида "*N* bochek" .
Например, "3 bochki" или "1 bochka" .
Напишите программу, которая выбирает правильное слово (из "bochka" , "bochek" , "bochki" )
в зависимости от *N*.
### Входные данные
Одно число *N* (0 ≤ *N* ≤ 1000).
### Выходные данные
Фраза на транслите (см. примеры).
### Примеры
**Входные данные:**
```
1
```
**Выходные данные:**
```
1 bochka
```
**Входные данные:**
```
15
```
**Выходные данные:**
```
15 bochek
```
**Входные данные:**
```
3
```
**Выходные данные:**
```
3 bochki
```
