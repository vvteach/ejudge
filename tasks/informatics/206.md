# Задача №206. Обход таблицы

В прямоугольной таблице *N*x*M* в начале игрок находится в левой верхней клетке. За один ход ему разрешается перемещаться в соседнюю клетку либо вправо, либо вниз (влево и вверх перемещаться запрещено). Посчитайте, сколько есть способов
у игрока попасть в правую нижнюю клетку.
### Входные данные
Вводятся два числа *N* и *M* - размеры таблицы (1<=*N*<=10, 1<=*M*<=10).
### Выходные данные
Выведите искомое количество способов.
***Примечание***
При указанных ограничениях число способов входит в тип Longint.
### Примеры
**Входные данные:**
```
1 10
```
**Выходные данные:**
```
1
```
