Считываем данные в две строки. Далее просто два вложенных цикла по всем символам первой и второй строк соответственно. Если совпадают и индексы и значения `bulls` увеличивается на единицу, если только значения, то `cows`.  
На Си тело цикла удобно оформить при помощи оператора **continue**:  
```
 if (c1[i] != c2[j]) continue;
 if (i == j) bulls++; else cows++;
```
