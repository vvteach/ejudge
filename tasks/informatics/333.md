# Задача №333. Четные числа

### Входные данные
Вводятся целые числа *a* и *b*. Гарантируется, что *a* не превосходит *b*
### Выходные данные
Выведите (через пробел) все четные числа от *a* до *b* (включительно).
### Примеры
**Входные данные:**
```
2
5
```
**Выходные данные:**
```
2 4
```
