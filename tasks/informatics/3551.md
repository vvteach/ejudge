# Задача №3551. Сумма факториалов

По данному натуральном $n$ вычислите сумму $1!+2!+3!+...+n!$.
В решении этой задачи можно использовать
только один цикл.
### Входные данные
Вводится натуральное число n.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
3
```
**Выходные данные:**
```
9
```
