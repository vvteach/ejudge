# Задача №315. Сумма квадратов

По данному натуральному *n* вычислите сумму 12+22+...+*n*2.
### Входные данные
Вводится единственное натуральное число *n*, не превосходящее 100
### Выходные данные
Необходимо вывести  вычисленную сумму.
### Примеры
**Входные данные:**
```
2
```
**Выходные данные:**
```
5
```
