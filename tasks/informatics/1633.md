# Задача №1633. Пицца

Вы решили заказать пиццу с доставкой на дом. Известно, что для клиентов, сделавших заказ на сумму более $C$ рублей, доставка является бесплатной, при заказе на $C$ рублей и меньше доставка стоит $B$ рублей.
Вы уже выбрали товара стоимостью $A$ рублей. В наличии имеются еще $N$ товаров стоимостью $d\_1, ..., d\_N$ рублей, каждый в единственном экземпляре. Их также можно включить в заказ.
Как потратить меньше всего денег и получить на дом уже выбранный товар стоимостью $A$ рублей?
### Входные данные
Вводятся сначала числа $A, B, C, N,$ а затем $N$ чисел $d\_1, ..., d\_N$. Все числа целые, $1 \le A \le 1000, 1 \le B ≤ 1000, 1 \le C \le 1000, 0 \le N \le 1000, 1 \le di \le 1 000 000$.
### Выходные данные
Выведите единственное число – суммарное количество денег, которое придется потратить.
### Примеры
**Входные данные:**
```
10 17 25
5
2 7 5 3 7
```
**Выходные данные:**
```
26
```
**Входные данные:**
```
100 1 50
5
5 2 4 3 1
```
**Выходные данные:**
```
100
```
**Входные данные:**
```
10 14 25
5
2 7 5 3 7
```
**Выходные данные:**
```
24
```
