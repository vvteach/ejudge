# Задача №3553. Потерянная карточка

Для настольной игры используются карточки с номерами от 1 до N.
Одна карточка потерялась. Найдите ее, зная номера оставшихся карточек.
### Входные данные
Дано число N, далее N-1 номер оставшихся карточек (различные числа от 1 до N).
### Выходные данные
Программа должна вывести номер потерянной карточки.
Для самых умных – массивами и аналогичными структурами данных
пользоваться нельзя.
### Примеры
**Входные данные:**
```
5
1
2
3
4
```
**Выходные данные:**
```
5
```
**Входные данные:**
```
4
3
2
4
```
**Выходные данные:**
```
1
```
