# Задача №2941. Последняя цифра

Дано натуральное число. Выведите его последнюю цифру.
### Входные данные
Вводится натуральное число.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
179
```
**Выходные данные:**
```
9
```
