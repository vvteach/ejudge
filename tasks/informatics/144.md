# Задача №144. N^N mod 10^P

### Входные данные
Даны два числа *N* и *P* (1≤*N*≤109, 1≤*P*≤100).
### Выходные данные
Вычислите *NN* mod 10*P*.
### Примеры
**Входные данные:**
```
5 3
```
**Выходные данные:**
```
125
```
