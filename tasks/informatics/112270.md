# Задача №112270. Заполнение, начиная с X

Напишите программу, которая заполняет массив из
*N*
элементов последовательными натуральными числами, начиная с
*X*
, и выводит его.
### Входные данные
Входная строка содержит два целых числа: начальное значение
*X*
и размер массива
*N*
, разделённые пробелом. Гарантируется, что
0 <
*N*
≤ 10000
.
### Выходные данные
Программа должна вывести содержимое массива:
*N*
последовательных натуральных чисел, начиная с
*X*
.
### Примеры
**Входные данные:**
```
4 5
```
**Выходные данные:**
```
4 5 6 7 8
```
