# Задача №111640. От списка ребер к спискам смежности

Простой неориентированный граф задан списком ребер, выведите его представление в виде списков смежности
### Входные данные
В первой строке входного файла заданы два целых числа
*N*
(
1 ≤
*N*
≤ 100
) — число вершин и
*M*
![](/moodle_probpics/111640/49e13cc29a32717673bd894ebde906fde8857d09.png)
— число ребер. Далее в
*M*
строках содержатся
*M*
пар чисел, каждая из которых описывает одно ребро графа. Гарантируется, что введённый список является корректным списком ребер некоторого неориентированного графа.
### Выходные данные
В выходной файл выведите матрицу смежности графа в следующем формате. Для каждой вершины выведите сначала количество смежных вершин, а потом перечислите их в этой же строке через пробел. В
*i*
-ой строке должно быть описание
*i*
-ой вершины.
### Примеры
**Входные данные:**
```
3 2
1 2
2 3
```
**Выходные данные:**
```
1 2
2 1 3
1 2
```
