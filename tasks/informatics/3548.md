# Задача №3548. Замечательные числа - 4

Даны два четырёхзначных числа A и B. Выведите все четырёхзначные числа на отрезке от A до B, запись которых
является палиндромом.
### Входные данные
Вводятся два целых числа *A* и *B*
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1600
2100
```
**Выходные данные:**
```
1661
1771
1881
1991
2002
```
