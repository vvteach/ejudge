# Задача №3547. Лесенка

По данному натуральному n≤9 выведите лесенку из n ступенек, i-я
ступенька состоит из чисел от 1 до i без пробелов.
### Входные данные
Вводится натуральное число.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
3
```
**Выходные данные:**
```
1
12
123
```
