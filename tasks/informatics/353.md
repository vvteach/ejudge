# Задача №353. Сумма степеней

### Входные данные
Вводится натуральное число *N*, которое не превосходит 30.
### Выходные данные
Вычислите 1+2+22+23+…+2 *N*.
Операцией возведения в степень пользоваться запрещено
### Примеры
**Входные данные:**
```
4
```
**Выходные данные:**
```
31
```
