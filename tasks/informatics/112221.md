# Задача №112221. Сумма и произведение

Напишите программу, которая вводит натуральное число
*N*
, затем –
*N*
целых чисел, и вычисляет их сумму и произведение.
### Входные данные
Первая строка содержит число
*N*
(
*N*
≥ 1
). В следующих
*N*
строках записаны целые числа, по одному в каждой строке.
### Выходные данные
Программа должна вывести в одной строке сумму и произведение введённых чисел, разделив их пробелом.
### Примеры
**Входные данные:**
```
5
1
2
3
4
5
```
**Выходные данные:**
```
15 120
```
