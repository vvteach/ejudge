# Задача №114335. Честное фото

У фермера Джона n ($1 \le n \le 100000$) коров. Они стоят в различных позициях вдоль длинной изгороди, $i$-я корова находится на позиции $x\_i$ ($ 0 \le x \le 10^9$) и имеет породу $b\_i$ ($1 \le b\_i \le 20$). Никакие две коровы не занимают
одну и ту же позицию. Джон хочет сделать фотографию непрерывного интервала коров, но так чтобы все породы коровы, представленные на фото были представлены одинаковым количеством коров. Фермер Паша также хочет чтобы по крайней мере $k$ ($1 \le k \le 20$) пород из 20 имеющихся были представлены на фото. Помогите фермеру Паше определить максимальный размер фото, удоволетворяющего требованиям.
Размер фото – это разница между максимальной и минимальной позициями коров на фото. Если не существует фото, удовлетворяющего всем ограничениям, выведите -1.
### Входные данные
Первая строка содержит числа $n$ и $k$, разделённые одним пробелом.
Следующие $n$ строк содержат описание коров: каждая строка содержит описание одной коровы двумя целыми числами, разделёнными одним пробелом: $x\_i$ и $b\_i$ - позицию коровы и её породу соответственно.
### Выходные данные
Выведите единственное целое число, – максимальный размер подходящего фото. Если таких фото нет, выведите -1.
### Примеры
**Входные данные:**
```
9 2
1 1
5 1
6 1
9 1
100 1
2 2
7 2
3 3
8 3
```
**Выходные данные:**
```
6
```
