# Задача №282. Биссектриса

### Входные данные
Шесть чисел – координаты точек X, Y и Z.
### Выходные данные
Три числа – коэффициенты уравнения биссектрисы угла YXZ.
### Примеры
**Входные данные:**
```
0 0 1 0 0 1
```
**Выходные данные:**
```
1.000000 -1.000000 0.000000
```
**Входные данные:**
```
0 0 1 3 1 -3
```
**Выходные данные:**
```
0.000000 -0.632456 0.000000
```
