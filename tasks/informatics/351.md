# Задача №351. Факториал

Вычислите *N*! ("эн-факториал") – произведение всех натуральных чисел от 1 до *N* ( *N*!=1∙2∙3∙…∙ *N* ).
### Входные данные
Вводится единственное число *N* – натуральное, не превосходит 12.
### Выходные данные
Выведите полученное значение *N*!
### Примеры
**Входные данные:**
```
3
```
**Выходные данные:**
```
6
```
