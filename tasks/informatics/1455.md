# Задача №1455. Разложение на простые

Требуется разложить целое число *N* на простые множители и вывести результат в
порядке возрастания.
### Входные данные
Программе дано число *N* (2 ≤ *N* ≤ 109).
### Выходные данные
Вывести разложение *N* на простые множители.
### Примеры
**Входные данные:**
```
2
```
**Выходные данные:**
```
2
```
**Входные данные:**
```
1008
```
**Выходные данные:**
```
2^4*3^2*7
```
