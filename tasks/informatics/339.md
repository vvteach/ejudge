# Задача №339. Минимальный делитель

Найдите самый маленький натуральный делитель числа *x*, отличный от 1 (2 <= *x* <= 30000).
### Входные данные
Вводится натуральное число *x*.
### Выходные данные
Выведите наименьший делитель числа *x*, отличный от 1.
### Примеры
**Входные данные:**
```
6
```
**Выходные данные:**
```
2
```
