# Задача №842. Последняя цифра числа Фибоначчи

Последовательность чисел Фибоначчи определяется следующим образом: *F0* = *F1* = 1,
*F**n+1* = *Fn*+*F n-1*. Напишите программу для вычисления *последней цифры n*-го члена последовательности.
### Входные данные
В единственной строке входных данных записано натуральное число *n* (1*≤n*≤1000).
### Выходные данные
Вывести последнюю цифру числа *Fn*.
### Примеры
**Входные данные:**
```
4
```
**Выходные данные:**
```
5
```
