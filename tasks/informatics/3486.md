# Задача №3486. 5E: удалите скобки

Дана строка, составленная из круглых скобок. Определите, какое наименьшее количество
символов необходимо удалить из этой строки, чтобы оставшиеся символы образовывали
правильную скобочную последовательность.
### Входные данные
Во входном файле записана строка из круглых скобок. Длина строки не превосходит ${100\,000}$ символов.
### Выходные данные
Выведите единственное целое число — ответ на поставленную задачу.
### Примеры
**Входные данные:**
```
())(()
```
**Выходные данные:**
```
2
```
**Входные данные:**
```
))(((
```
**Выходные данные:**
```
5
```
