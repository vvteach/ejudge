# Задача №3153. Четные элементы

Выведите все четные элементы списка.
### Входные данные
Вводится список чисел. Все числа списка находятся на одной строке.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1 2 2 3 3 3 4
```
**Выходные данные:**
```
2 2 4
```
