# Задача №3457. Следующее и предыдущее

Напишите программу, которая считывает целое число и выводит текст, аналогичный приведенному
в примере (пробелы важны!):
### Входные данные
Вводится целое число (гарантируется, что число находится в диапазоне от -1000 до +1000).
### Выходные данные
### Примеры
**Входные данные:**
```
179
```
**Выходные данные:**
```
The next number for the number 179 is 180.
The previous number for the number 179 is 178.
```
