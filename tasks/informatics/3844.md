# Задача №3844. Удалить элемент

Дан список из чисел и индекс элемента в списке k. Удалите из списка элемент с индексом k,
сдвинув влево все элементы, стоящие правее элемента с индексом k.
### Входные данные
Вводится список чисел. Все числа списка находятся на одной строке. В следующей строке вводится одно целое число.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
7 6 5 4 3 2 1
2
```
**Выходные данные:**
```
7 6 4 3 2 1
```
