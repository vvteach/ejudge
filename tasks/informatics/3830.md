# Задача №3830. Количество положительных

Найдите количество положительных ($> 0$) элементов в данном списке.
### Входные данные
Вводится список чисел. Все числа списка находятся на одной строке.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1 -2 3 -4 5
```
**Выходные данные:**
```
3
```
