# Задача №274. Принадлежность точки прямой

### Входные данные
Пять чисел – координаты точки и коэффициенты A, B и C уравнения прямой.
### Выходные данные
Одна строка “YES”, если точка принадлежит прямой, и “NO” в противном случае.
### Примеры
**Входные данные:**
```
8 7 2 -3 5
```
**Выходные данные:**
```
YES
```
