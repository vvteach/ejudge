# Задача №3476. Дележ яблок - 3

$n$ школьников делят $k$ яблок “поровну”, то есть
так, чтобы количество яблок, доставшихся любым двум школьникам, отличалось
бы не более, чем на 1.
### Входные данные
Программа получает на вход числа
$n$ и $k$.
### Выходные данные
Программа должна вывести количество школьников, которым достанется
яблок меньше, чем некоторым из их товарищей.
### Примеры
**Входные данные:**
```
7
30
```
**Выходные данные:**
```
5
```
