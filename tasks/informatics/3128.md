# Задача №3128. Вставка символов

Дана строка. Получите новую строку, вставив между двумя символами исходной строки символ \*. Выведите полученную строку.
### Входные данные
Вводится строка.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
Python
```
**Выходные данные:**
```
P*y*t*h*o*n
```
