# Задача №270. Уравнение прямой I

### Входные данные
Четыре числа – координаты двух различных точек на прямой.
### Выходные данные
Три числа – коэффициенты A, B и C уравнения этой прямой.
### Примеры
**Входные данные:**
```
1 0 2 1
```
**Выходные данные:**
```
1 -1 -1
```
