# Задача №1518.

Вкладчик положил на банковский счет n рублей. Каждый год на сумму вклада начисляется k процентов годовых (будем считать, что процент всегда округляется до целого числа рублей по формуле [xk/100], где x — сумма вклада на начало года). Начисленные
проценты добавляются к сумме вклада. Через сколько лет сумма вклада станет не менее m рублей?
### Входные данные
На вход программе подаются три натуральных числа: $n \le 10^6, k \le 100, m \le 1000\*n $.
### Выходные данные
Выведите одно число — искомое количество лет.
### Примеры
**Входные данные:**
```
100 10 111
```
**Выходные данные:**
```
2
```
**Входные данные:**
```
100 1 100
```
**Выходные данные:**
```
0
```
