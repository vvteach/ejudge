# Задача №112506. Считалка

Чтобы выбрать водящего в игре,
*N*
ребят встают в круг и, начав отсчет от первого, удаляют каждого
*K*
-ого, смыкая круг после каждого удаления. Определить порядок удаления ребят из круга, то есть напечатать номера ребят в том порядке, как они удаляются из круга.
### Входные данные
Входная строка содержит числа
*N*
(
1 ≤
*N*
≤ 10000
) и
*K*
(
1 ≤
*K*
≤ 10000
), разделённые пробелом.
### Выходные данные
Программа должна вывести в одну строку номера ребят в том порядке, как они удаляются из круга, разделив их пробелами.
### Примеры
**Входные данные:**
```
5 3
```
**Выходные данные:**
```
3 1 5 2 4
```
