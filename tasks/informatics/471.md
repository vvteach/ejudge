# Задача №471. Степени вершин по спискам ребер

Неориентированный граф задан списком ребер. Найдите степени всех вершин графа.
### Входные данные
Сначала вводятся числа *n* (
$1 \le n \le 100$
) – количество вершин в графе и *m* (
$1 \le m \le n(n - 1) /2$
) – количество ребер. Затем следует *m* пар чисел – ребра графа.
### Выходные данные
Выведите *n* чисел – степени вершин графа.
### Примеры
**Входные данные:**
```
5 3
1 3
2 3
2 5
```
**Выходные данные:**
```
1
2
2
0
1
```
