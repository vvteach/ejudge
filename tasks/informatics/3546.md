# Задача №3546. Замечательные числа - 3

Дано натуральное число n. Выведите в порядке возрастания
все трехзначные числа, сумма цифр которых равна n.
### Входные данные
Вводится натуральное число *n*.
Выведите ответ на задачу.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
3
```
**Выходные данные:**
```
102
111
120
201
210
300
```
