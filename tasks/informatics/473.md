# Задача №473. Полустепени вершин по спискам ребер

Ориентированный граф задан списком ребер. Найдите степени всех вершин графа.
### Входные данные
Сначала вводятся числа *n* (
$1 \le n \le 100$
) –  количество вершин в графе и *m* (
$1 \le m \le n(n - 1)$
) – количество ребер. Затем следует *m* пар чисел – ребра графа.
### Выходные данные
Выведите  *n* пар чисел – для каждой вершины сначала выведите полустепень захода и затем полустепень исхода.
### Примеры
**Входные данные:**
```
5 3
2 5
3 1
3 2
```
**Выходные данные:**
```
1
0
1
1
0
2
0
0
1
0
```
