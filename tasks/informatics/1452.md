# Задача №1452. Целые точки отрезка

Требуется написать программу, которая вычислит, сколько всего точек с целочисленными
координатами принадлежат отрезку.
### Входные данные
Даны четыре целых числа – координаты концов отрезка (*x*1, *y*1) и (*x*2, *y*2).
Каждая из координат не превышает по абсолютной величине значения 1000.
### Выходные данные
Требуется вывести количество точек отрезка, имеющих целочисленные координаты.
### Примеры
**Входные данные:**
```
1 0 5 0
```
**Выходные данные:**
```
5
```
**Входные данные:**
```
-1 -2 2 4
```
**Выходные данные:**
```
4
```
