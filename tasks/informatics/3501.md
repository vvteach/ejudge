# Задача №3501. Максимум двух чисел

Даны два целых числа. Выведите значение наибольшего из них.
### Входные данные
Вводятся два числа.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
5
3
```
**Выходные данные:**
```
5
```
**Входные данные:**
```
3
3
```
**Выходные данные:**
```
3
```
