# Задача №269. Площадь треугольника

### Входные данные
Шесть чисел – координаты трёх вершин треугольника.
### Выходные данные
Одно число – величина площади треугольника.
### Примеры
**Входные данные:**
```
1 1 2 4 3 2
```
**Выходные данные:**
```
2.50000
```
