# Задача №1411. Сортировка пузырьком

Определите, сколько обменов сделает алгоритм пузырьковой сортировки по возрастанию для данного массива.
### Входные данные
На первой строке дано число *N* (1 ≤ *N* ≤ 1000) – количество элементов в массиве.
На второй строке – сам массив. Гарантируется, что все элементы массива различны и не превышают
по модулю 109.
### Выходные данные
Выведите одно число – количество обменов пузырьковой сортировки.
### Примеры
**Входные данные:**
```
5
1 2 3 4 5
```
**Выходные данные:**
```
0
```
**Входные данные:**
```
5
5 4 3 2 1
```
**Выходные данные:**
```
10
```
