# Задача №3744. Замена подстроки

Дана строка. Замените в этой строке все цифры `1` на слово
`one`.
### Входные данные
Вводится строка.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1+1=2
```
**Выходные данные:**
```
one+one=2
```
