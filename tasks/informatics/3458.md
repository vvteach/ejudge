# Задача №3458. Дележ яблок - 1

$n$ школьников делят $k$ яблок поровну, неделящийся остаток остается в корзинке.
Сколько яблок достанется каждому школьнику?
### Входные данные
Программа получает на вход числа
$n$ и $k$ - целые, положительные, не превышают 10000.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
3
14
```
**Выходные данные:**
```
4
```
