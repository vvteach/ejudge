# Задача №3510. Ход короля

Шахматный король ходит по горизонтали, вертикали и диагонали, но только на 1 клетку. Даны две различные клетки шахматной доски, определите,
может ли король попасть с первой клетки на вторую одним ходом.
В этой задаче удобно использовать функцию abs(x), которая соответствует операции |x|.
### Входные данные
Программа получает на вход четыре числа
от 1 до 8 каждое, задающие номер столбца и номер строки сначала для первой клетки, потом для второй
клетки.
### Выходные данные
Программа должна вывести `YES`, если из первой клетки ходом короля можно попасть во вторую или `NO` в противном случае.
### Примеры
**Входные данные:**
```
4
4
5
5
```
**Выходные данные:**
```
YES
```
