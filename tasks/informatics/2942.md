# Задача №2942. Число десятков двузначного числа

Дано двузначное число. Найдите число десятков в нем.
### Входные данные
Вводится двухзначное число.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
42
```
**Выходные данные:**
```
4
```
