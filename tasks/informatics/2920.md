# Задача №2920. Перекраска полоски

Дана
полоска *Nx*1 клетку, каждая клетка которой раскрашена в один из
*M* цветов. За один ход разрешается перекрасить непрерывную
область одного цвета в любой другой цвет.
Требуется
определить наименьшее число перекрашиваний, за которое можно получить
полоску одного (любого) цвета.
### Входные данные
В
первой строке находятся два числа *N* и *M* – ширина
полоски и количество цветов соответственно. 1 ≤
*N* ≤ 100, 1 ≤
*M* ≤
100. Во второй строке находятся *N* чисел, соответствующих
цветам каждой из клеток полоски от 1 до *N*
(сами цвета лежат в диапазоне от 1 до *M*,
каждый цвет встречается хотя бы один раз).
### Выходные данные
Выведите
одно число – минимальное число перекрашиваний, за которое можно
получить полоску одного цвета.
### Примеры
**Входные данные:**
```
5 3
3 2 1 1 3
```
**Выходные данные:**
```
2
```
