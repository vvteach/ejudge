# Задача №3464. Сумма цифр

Дано трехзначное число. Найдите сумму его цифр.
### Входные данные
Вводится целое положительное число. Гарантируется, что оно соответствует условию задачи.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
179
```
**Выходные данные:**
```
17
```
