# Задача №1447. Контроперация

Хакер Василий получил доступ к классному журналу и хочет заменить все свои минимальные оценки
на максимальные. Напишите программу, которая заменяет оценки Василия, но наоборот (все максимальные - на минимальные).
### Входные данные
Дано количество оценок Василия (не больше 100), затем сами оценки.
### Выходные данные
Требуется вывести исправленные оценки в том же порядке.
### Примеры
**Входные данные:**
```
5 1 3 3 3 4
```
**Выходные данные:**
```
1 3 3 3 1
```
**Входные данные:**
```
8 5 4 2 2 4 2 2 5
```
**Выходные данные:**
```
2 4 2 2 4 2 2 2
```
