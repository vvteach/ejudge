# Задача №3058. Минимальный делитель

Дано целое число $N$, не меньшее 2. Выведите его наименьший натуральный делитель, отличный от 1.
### Входные данные
Вводится натуральное число \(2 \le N \ 1000).
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
15
```
**Выходные данные:**
```
3
```
