# Задача №3804. Сумма последовательности

Дана последовательность чисел, завершающаяся числом 0. Найдите сумму всех этих чисел, не используя
цикл.
### Входные данные
Вводится последовательность целых чисел, оканчивающаяся числом 0 (само число 0 в последовательность не входит, а служит как признак ее окончания).
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1
7
9
0
```
**Выходные данные:**
```
17
```
