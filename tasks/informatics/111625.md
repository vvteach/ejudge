# Задача №111625. Почти простое число

Назовем натуральное число почти простым, если оно раскладывается на произведение каких-нибудь двух неравных простых.
### Входные данные
Дано одно натуральное число
*n*
(
2 ≤
*n*
≤ 2·10
9
).
### Выходные данные
Выведите «YES», если
*n*
почти простое. Иначе выведите «NO»
### Примеры
**Входные данные:**
```
10
```
**Выходные данные:**
```
YES
```
**Входные данные:**
```
9
```
**Выходные данные:**
```
NO
```
