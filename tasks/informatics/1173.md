# Задача №1173. Удаление цифр

Задана строка S (длина не превышает 255 символов). Удалите из нее все цифры 1 и 2 и выведите результат.
### Входные данные
В первой строке файла записана строка S
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
2Formula_1!3
```
**Выходные данные:**
```
Formula_!3
```
