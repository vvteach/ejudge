# Задача №3531. Сумма квадратов

По данному натуральном $n$ вычислите сумму
$1^2+2^2+3^2+...+n^2$.
### Входные данные
Вводится натуральное число.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
3
```
**Выходные данные:**
```
14
```
