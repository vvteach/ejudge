# Задача №3163. Вывести в обратном порядке

Выведите элементы данного списка в обратном порядке, не изменяя сам список.
### Входные данные
Вводится список чисел. Все числа списка находятся на одной строке.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1 2 3 4 5
```
**Выходные данные:**
```
5 4 3 2 1
```
