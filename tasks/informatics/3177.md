# Задача №3177. Большой сдвиг

Дан список из $N$ ($1 \le N \le 100000$) целых чисел и число $K$
($|K| < 100000 $).
Циклически сдвиньте список на $|K|$ элементов вправо,
если $K$ – положительное и влево, если отрицательное
число.
### Входные данные
Программа получает на вход список целых чисел, затем число $K$.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
5 3 7 4 6
3
```
**Выходные данные:**
```
7 4 6 5 3
```
