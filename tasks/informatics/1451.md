# Задача №1451. Четные и нечетные числа

Даны три целых числа *A*, *B*, *C*.
Определить, есть ли среди них хотя бы одно четное и хотя бы одно нечетное.
### Входные данные
Числа *A*, *B*, *C*, не превышающие по модулю 10000.
### Выходные данные
Одна строка – "YES" или "NO".
### Примеры
**Входные данные:**
```
3
4
5
```
**Выходные данные:**
```
YES
```
**Входные данные:**
```
7
7
7
```
**Выходные данные:**
```
NO
```
