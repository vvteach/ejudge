# Задача №844. Хитрая последовательность

Вычислите n-й член последовательности, заданной формулами:
a2n = an ­+ 1 (n>0),   
a2n+2 = a2n+1 - an (n>0),
a0=1, a1=1.
### Входные данные
Вводится натуральное число n, не превосходящее 1000.
### Выходные данные
Выведите ответ к задаче.
### Примеры
**Входные данные:**
```
4
```
**Выходные данные:**
```
3
```
