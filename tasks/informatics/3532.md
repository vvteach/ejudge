# Задача №3532. Сумма кубов

По данному натуральном n вычислите сумму
$1^3+2^3+3^3+...+n^3$.
### Входные данные
Вводится число *n*.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
1
```
**Выходные данные:**
```
1
```
**Входные данные:**
```
3
```
**Выходные данные:**
```
36
```
