# Задача №468. Проверка на наличие параллельных ребер, неориентированный вариант

Неориентированный граф задан списком ребер. Проверьте, содержит ли он параллельные ребра.
### Входные данные
Сначала вводятся  числа *n* (
$1 \le n \le 100$
) – количество вершин в графе и *m* (
$1 \le m \le 10,000$
) – количество ребер. Затем следует *m* пар чисел – ребра графа.
### Выходные данные
Выведите  «YES», если граф содержит параллельные ребра, и «NO» в противном случае.
### Примеры
**Входные данные:**
```
5 3
1 3
2 3
2 5
```
**Выходные данные:**
```
NO
```
