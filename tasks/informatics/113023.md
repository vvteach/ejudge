# Задача №113023. Количество максимумов

Требуется за один проход по массиву определить максимальное значение и количество раз, которое это значение встречается в массиве.
### Входные данные
На вход подаётся натуральное число $N$ ($N \le 10^5$).
Затем на вход подаётся строка, в которой через пробел записаны $N$ целых чисел.
### Выходные данные
Программа должна вывести два числа, разделённые пробелом: максимум и их количество.
### Примеры
**Входные данные:**
```
7
8 7 7 5 5 8 5
```
**Выходные данные:**
```
8 2
```
