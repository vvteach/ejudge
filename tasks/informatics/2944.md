# Задача №2944. Сумма цифр

Дано **трехзначное** число. Найдите сумму его цифр.
### Входные данные
Вводится трехзначное число.
### Выходные данные
Выведите ответ на задачу.
### Примеры
**Входные данные:**
```
179
```
**Выходные данные:**
```
17
```
