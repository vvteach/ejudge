# Задача №1430. ГНЧЭ-1

"ГНЧЭ-1"  – сложное электронное устройство, выдающее каждую секунду очередное число
последовательности 1, 2, 2, 3, 3, 3, 4, 4, 4, 4, 5... Ввиду дороговизны электронных
комплектующих вам поручено разработать эмулятор для этого устройства.
### Входные данные
Дано количество секунд (от 1 до 1000000), которые работает генератор после включения.
### Выходные данные
Результат работы генератора
### Примеры
**Входные данные:**
```
2
```
**Выходные данные:**
```
1 2
```
**Входные данные:**
```
5
```
**Выходные данные:**
```
1 2 2 3 3
```
