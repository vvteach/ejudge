# Задача №111642. Сумма двух чисел

### Входные данные
На первой строке входного файла находятся два целых числа
*a*
и
*b*
(
- 10
9
≤
*a*
,
*b*
≤ 10
9
).
### Выходные данные
Вашей программе требуется вывести единственное число — сумму заданных чисел
*a*
+
*b*
.
### Примеры
**Входные данные:**
```
2 3
```
**Выходные данные:**
```
5
```
**Входные данные:**
```
17 -18
```
**Выходные данные:**
```
-1
```
